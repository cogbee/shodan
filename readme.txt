http://www.shodanhq.com/
这个搜索引擎重要之处不需多说。对服务器基础服务的扫描，可以获取很多服务，比如该服务器是否有什么协议在运行等等。
但是这个搜索引擎有一个问题，就是用户必须注册，注册用户只能看前5页的数据，除非你购买。

对于没有买的用户，改程序是一个突破。

原理：
http://www.shodanhq.com/host/view/175.102.10.175
可以得到175.102.10.175主机的shodan搜索得到的结果。当然有没有更加高级的结果我就不知道了。
改程序一直换IP地址，然后就得到不一样的主机的结果，搜索的服务主要有：
['MySQL','NetBIOS','HTTP','SSH','FTP','RDP','MongoDB','POP3 + SSL','IMAP + SSL','Redis','PostgreSQL','DNS','SMTP','MS-SQL']

该程序必须需要用户登录之后的cookie，这个cookie我是用的我自己的内嵌在程序中的。这个没有单独拿出来当一个参数了。如果我的cookie失效，就需要用户自己的cookie了。

得到的结果保存在文件中。
